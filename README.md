# 0의 집행 - Choose Your Own Adventure

🕵️ **사이버펑크 노아르 미스터리 게임**

## 🎮 게임 소개

"0의 집행"은 Choose Your Own Adventure 스타일의 인터랙티브 미스터리 게임입니다.
강박사의 의문사 사건을 세 가지 완전히 다른 관점에서 해결해보세요.

### 🎯 특징

- **3개의 고유한 스토리 경로**
- **각 경로별 15개 장면** (총 51개 이미지)
- **일본식 비주얼 노벨 UI**
- **분기별 선택지 시스템**
- **완전히 다른 3가지 엔딩**

## 🔍 스토리 경로

### 🕵️ Detective Path - "논리의 승리"
과학적 분석과 논리적 추론으로 사건을 해결하는 클래식 미스터리 경로

### 🎭 Conspiracy Path - "거대한 그림자"
정부-기업 음모를 파헤치는 정치 스릴러 경로

### 💔 Human Drama Path - "마음의 진실"
인간관계와 감정에 집중한 휴먼 드라마 경로

## 🎮 플레이 방법

1. **게임 시작**: 공통 도입부 4개 장면
2. **경로 선택**: 수사 방향에 따라 3개 경로 중 선택
3. **분기 탐험**: 각 경로 내에서 추가 선택지
4. **엔딩 도달**: 경로별 고유한 결말
5. **다시 플레이**: 다른 경로로 새로운 스토리 체험

## 🚀 빠른 시작

- **스토리 경로**: 3개
- **선택지 수**: 12개
- **플레이 시간**: 경로당 15-20분
