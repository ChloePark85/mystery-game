# 🕵️ Detective Path - "논리의 승리" 스크립트

## Phase 1: 과학적 분석

### detective_01 - 섬유 분석
**진**: "현미경으로 로봇 팔에서 발견된 섬유를 분석해보자. 이 작은 조각이 사건의 열쇠가 될 수 있어."
**진**: "흥미롭군. 이 섬유의 직조 방식은... 고급 실크 넥타이다. 강박사가 착용했던 바로 그 넥타이와 일치해."

### detective_02 - 3D 분석
**진**: "로봇 팔의 구조를 3D로 분석해보자. 물리학적 메커니즘을 정확히 이해해야 해."
**진**: "이 관절 부분... 중력에 의해 자동으로 닫히는 구조로 되어 있어. 21시에 전원이 차단되면서..."

### detective_03 - 심박수 데이터
**진**: "유지민의 스마트워치 데이터를 타임라인과 대조해보자."
**진**: "20시 55분에 심박수가 급상승했어. 바로 그 시간이... 강박사가 사망한 추정 시각이야."

### detective_04 - 추론 과정
**진**: "모든 증거를 연결해보자. 물리적 증거, 시간, 그리고 동기..."
**진**: "유지민이 로봇 팔 근처에서 작업하다가 실수로... 아니다, 이건 우발적 사고가 아니야."

### detective_05 - 법정 승리
**진**: "존경하는 재판장님, 과학적 증거는 명확합니다. 피고인의 유죄를 증명하는..."
**진**: "물리학은 거짓말을 하지 않습니다. 이 사건의 진실은 바로 이것입니다."

## Phase 2: 심층 분석

### detective_06 - 포렌식 소프트웨어
**진**: "고급 분석 소프트웨어로 3D 재구성을 해보자. 모든 각도를 정밀하게..."
**진**: "여기야! 이 지점에서 넥타이가 걸렸고, 중력의 힘으로..."

### detective_07 - UV 조사
**진**: "UV 라이트로 숨겨진 흔적을 찾아보자."
**진**: "봐라! 여기에 지워진 지문이 있어. 누군가가 의도적으로 증거를 조작했어."

### detective_08 - 증거 정리
**진**: "모든 증거를 체계적으로 정리했어. 시간순으로 배열하면..."
**진**: "패턴이 보이기 시작해. 이건 계획된 범죄야."

### detective_09 - 실험 재현
**진**: "실제로 같은 조건에서 실험해보자. 과학적 검증이 필요해."
**진**: "결과가 나왔어. 우발적 사고로는 이런 결과가 나올 수 없어."

### detective_10 - 최종 분석
**진**: "모든 데이터를 종합한 결과... 범인은 명확해."
**진**: "과학은 진실을 말하고 있어. 더 이상 숨길 수 없어."

## Phase 3: 진실 규명

### detective_11 - 논리적 압박
**진**: "증거는 명확해요, 유지민씨. 과학적 분석 결과를 부정할 수 있나요?"
**유지민**: "그건... 그건 사고였어요!"
**진**: "사고? 21시 정확히 시스템을 종료시킨 것도 사고였나요?"

### detective_12 - 메커니즘 설명
**진**: "이 로봇 팔은 전원이 차단되면 중력에 의해 자동으로 닫혀요."
**진**: "당신은 이 메커니즘을 알고 있었고, 의도적으로 이용했어요."

### detective_13 - 보안 영상 분석
**진**: "보안 카메라 영상을 프레임 단위로 분석했어요."
**진**: "20시 54분, 당신이 강박사의 넥타이를 로봇 팔에 끼워 넣는 모습이 보여요."

### detective_14 - 최종 증거 제시
**진**: "모든 증거가 당신을 가리키고 있어요. 과학적 분석, 타임라인, 동기까지."
**유지민**: "더 이상... 숨길 수 없겠네요."

### detective_15 - 승리
**진**: "진실은 언제나 과학적 방법으로 밝혀진다. 정의가 승리했어."
**진**: "또 하나의 사건이 논리와 증거로 해결되었군."
