## 이미지 생성 시 주의사항

1. **일관된 캐릭터**: 같은 인물이 여러 장면에 등장할 때 외모가 일관되도록 주의
2. **한국적 배경**: 서울의 도시 풍경, 한국인 캐릭터의 특징을 반영
3. **사이버펑크 노아르**: 어두운 톤의 미래적 분위기와 하드보일드 감성
4. **고해상도**: 게임 UI에 적합한 선명한 이미지 (1920x1080 추천)
5. **16:9 비율**: 게임 화면에 최적화된 가로형 비율

## 파일 저장 규칙

- 파일명: `scene_01.png`, `scene_02.png` ... `scene_25.png`
- 저장 위치: `game/images/scenes/` 폴더
- 포맷: PNG (투명 배경 불필요)
- 크기: 1920x1080 또는 유사한 16:9 비율
